# AI Stock & Crypto Predictor

This project uses Machine Learning to predict stock and cryptocurrency prices.

## Features
- Bitcoin price prediction
- Machine learning model
- Data visualization
- Easy to run

## Technologies
Python
Scikit-learn
Pandas
Matplotlib
Yahoo Finance API

## Installation

pip install -r requirements.txt

## Run

python train_model.py
python predictor.py

## Example Output

The model predicts future prices based on historical data.
